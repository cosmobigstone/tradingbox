FAQ

자주 묻는 질문

내 자산이 안전한가요? ▼
TradingBox는 출금 권한이 없는 API Key만 사용합니다. 고객의 자산에 직접 접근하거나 출금할 수 없습니다. 언제든 API Key를 거래소에서 삭제하여 즉시 연동을 해제할 수 있습니다.
손실이 나면 어떻게 되나요? ▼
서킷브레이커가 자동으로 작동합니다. 일일 손실이 설정 한도(기본 5%)에 도달하면 모든 포지션이 자동 청산되고 당일 거래가 중단됩니다. ATR 기반 동적 손절매도 항상 활성화되어 단일 포지션 손실을 제한합니다.
어떤 거래소를 지원하나요? ▼
현재 Binance(코인 선물), Alpaca(미국 주식)를 지원합니다. OKX, Bybit, 국내 거래소는 로드맵에 포함되어 있으며 순차 추가됩니다.
수익률을 보장하나요? ▼
금융 상품의 특성상 수익률을 보장하지 않습니다. 백테스트 및 페이퍼 트레이딩 기반 목표 수익률은 제시되나, 실제 시장 환경에 따라 결과는 달라질 수 있습니다. 원금 손실 가능성을 인지하고 투자하시기 바랍니다.
언제든 해지할 수 있나요? ▼
예, 언제든 서비스를 해지할 수 있습니다. 해지 즉시 모든 자동 거래가 중단됩니다. 첫 14일 내 해지 시 전액 환불해 드립니다.
